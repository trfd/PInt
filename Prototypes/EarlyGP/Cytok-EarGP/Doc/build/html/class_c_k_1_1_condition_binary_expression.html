Detailed Description

Binary relationship between to ICondition. These are created when using & (&&) or | (||) operators on ICondition. The ConditionBinaryCompound keep a reference to the operand. In this way the expression creation and evaluation can be split in times. This also allows recurent evaluation. Both operand are evaluated ever since the value of the first operand could give a result. This allow user implementing its own conditions to be sure that the evaluation code is processed every evaluation.
