Method to override when creating a custom ICondition. This method must return the value of the condition. No assumptions on the usage of Evaluate should be made in order to ensure the most flexible usage. Mainly Evaluate is called when the value of the condition is retrieved (either by the getter ICondtion.value, by a cast to a boolean or by assigment to a bool).
