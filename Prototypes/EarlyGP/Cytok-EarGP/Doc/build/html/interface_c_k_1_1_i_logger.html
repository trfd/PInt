Detailed Description

Logger interface. Logger is the subsystem that actually write something somewhere when a CK.Log method is called. The most basic Logger is the one that writes into .NET's console. Enhanced debugger can be create such as HTML or ingame console logs. To change the logger simply set the value of CK.Log.logger to an instance of your custom logger implementation.
